# 💞 Nosso Momento

**Nosso Momento** é um app web para casais que permite a conexão entre parceiros através de um sistema de pareamento por número de telefone. Após o pareamento, os casais podem interagir em um ambiente seguro e compartilhado.

## ✨ Funcionalidades

- Cadastro e login com autenticação via Firebase
- Pareamento entre casais por número de telefone
- Salvamento de dados em tempo real no Firestore
- Interface leve, acessível e responsiva

## 🛠 Tecnologias utilizadas

- **HTML5** + **CSS3**
- **JavaScript Vanilla (puro)**
- **Firebase** (Authentication, Firestore)
- **Vercel** (Hospedagem)
